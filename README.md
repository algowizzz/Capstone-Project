# Excel Capstone Project – Healthcare Data Analysis

## Problem Statement
Healthcare organizations generate large volumes of data, but inconsistencies, missing values, and unstructured fields make analysis challenging.  
This project focuses on cleaning, analyzing, and visualizing healthcare data to identify meaningful trends related to:
- Patient demographics  
- Treatment outcomes  
- Operational efficiency  

---

## Dataset Overview
The dataset contains:
- Patient demographic details  
- Admission and discharge records  
- Diagnosis and treatment information  
- Cost-related variables  

It reflects real-world data challenges such as:
- Missing values  
- Inconsistent date formats  
- Ambiguous categorical data  

---

## Data Cleaning & Preparation
Performed comprehensive data preprocessing to ensure accuracy and usability:

- Handled missing and null values using appropriate imputation techniques  
- Standardized and corrected inconsistent date formats  
- Removed duplicate and irrelevant records  
- Normalized categorical variables (diagnosis, treatment type, patient category)  
- Created derived columns to enhance analytical insights  

---

## Analysis Performed
Conducted exploratory data analysis to extract key insights:

- Patient demographic analysis (age groups and gender distribution)  
- Admission and discharge trends over time  
- Treatment-wise outcome comparison  
- Identification of high-risk and high-cost patient segments  

---

## Tools & Technologies
- Microsoft Excel  
  - Advanced Formulas  
  - Pivot Tables  
  - Data Cleaning Techniques  
  - Interactive Dashboards  

---

## Key Insights
- Certain age groups exhibited higher average treatment costs and longer hospital stays  
- Specific treatments were associated with extended recovery periods  
- Identified patterns that highlight opportunities for operational optimization  

---

## Conclusion
This project demonstrates end-to-end data analytics capabilities, including:
- Data cleaning and preprocessing  
- Exploratory data analysis (EDA)  
- Dashboard creation and visualization  

It highlights the ability to transform raw healthcare data into actionable insights that support data-driven decision-making.
